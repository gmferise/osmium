About Osmium

Purpose

Osmium is a user-friendly interface for using Google Sheets as a database.
It was created as a school project for the Hamilton Southeastern Highschool Library in the Software Development course.
Its main use is for the library to monitor traffic flow through the library and common uses of the library.
Touch-screen kiosks are set up at each entrance to the library and students are asked to use their student id to sign in and out of the library.
Librarians have the opportunity to assist students seeking technology help, sending them to the building techs if they are unable to resolve the issue.

Developers
